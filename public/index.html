GitHub Activity Art
Настройте свой график активности GitHub так, как вам нравится, используя этот инструмент для рисования. Он генерирует набор коммитов, который заполняет график активности.

Публикация

  • Создайте репозиторий на GitHub.
  • Склонируйте репозиторий: 
    git clone https://github.com/{USERNAME}/{REPOSITORY NAME}.git
  • Запустите сгенерированный скрипт.
  • Занесите все изменения, которые были сделаны в локальном репозитории, на GitHub: 
    git push -u origin master
